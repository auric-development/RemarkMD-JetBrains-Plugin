# RemarkMD sample

This is a sample Markdown document for trying the RemarkMD preview inside a JetBrains IDE.

## How to use it

Select a passage of text in the **preview pane** on the right, then click **Add Comment**.
The comment is written into this file as YAML front matter, and a `[^cN]` marker is inserted
inline right here in the prose.

## A longer paragraph

Reviewers leave threaded comments anchored to the text, and Claude answers them by revising the
prose and marking each comment resolved. The file itself is the database and the handoff protocol,
so there is no server and no API call involved.
